# 👨‍💼 Employee Management System (Python)

A simple **Object-Oriented Python project** that allows users to manage employee records from the command line.
This system supports creating and displaying different types of employees such as **Employee, Manager, and Developer**.

---

## 🚀 Features

✔️ Add basic Employee details
✔️ Add Manager with department
✔️ Add Developer with programming language
✔️ Display all stored records
✔️ Simple menu-driven interface
✔️ Built using OOP concepts

---

## 🛠️ Technologies Used

* Python 3.x
* Object-Oriented Programming (OOP)
* Inheritance & Method Overriding

## ▶️ How to Run

1. Install Python (if not installed)
2. Download or clone this repository
3. Open terminal / command prompt
4. Run the program:

```
python main.py
```
